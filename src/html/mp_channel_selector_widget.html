Channels
    GIVEN a user is in Social Posts
    AND they want to change the default copy
    WHEN they click on the "Post" field
    THEN enable them to edit the post copy
    AND save changes in real time
    AND apply the edited copy to all post channels scheduled for that day

    GIVEN a user is editing the copy of a multi-post
    AND they leave the Post copy field blank
    WHEN they click out of the Post copy field
    THEN show the following error message: "This field is required"
